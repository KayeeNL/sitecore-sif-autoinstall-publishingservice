This folder is used to store the following:
- Copy over the Sitecore Publishing Service and Sitecore Publishing Module zip files from dev.sitecore.net to this SitecorePackages directory.

UPDATE: From SPS version 6.0.0 (for 10.2) on you need a valid Sitecore license when using the Publishing Service. 
- Copy over a valid Sitecore license to this SitecorePackage folder and the script will take care of the correct install.
